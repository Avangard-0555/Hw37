Установите зависимости командой:

pip install -r requirements.txt

Шаг 4: Запуск приложения
	1.	В терминале перейдите в папку проекта:

cd /path/to/text_to_speech_project


	2.	Запустите приложение:

python app.py


	3.	Откройте браузер и перейдите по адресу http://127.0.0.1:5000/.

Дополнительные шаги (опционально)
	1.	Стилизация интерфейса: добавьте CSS-файл в папку static и подключите его в index.html.
	2.	Развёртывание приложения: для публикации используйте платформы, такие как Heroku, Render или Vercel.
	3.	Поддержка высококачественного TTS: интегрируйте API, например, Google Cloud Text-to-Speech (могу помочь с этим, если потребуется)
